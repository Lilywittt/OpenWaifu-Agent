# Source

这里放后续产品代码。代码边界按 B 端技术美术生产流程划分。

```text
assets/     角色、服装、表情、参考图和模型资源
tasks/      美术任务、批量任务和任务状态
workflows/  ComfyUI 与其他生成后端适配
review/     审图、收藏、弃用、问题标注
export/     图片规格导出、命名和入库
feedback/   反馈数据沉淀、训练数据候选整理
handoff/    ChatGPT Plus 网页手动生产包、导入结果和包记录
api/        后端服务接口
client/     前端客户端
plugins/    可插拔后端、导入器和导出器
```
