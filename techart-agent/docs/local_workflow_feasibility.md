# 本地工作流可行性判断

本文件用于判断产品设计能否被本地 ComfyUI 工作流支撑。核心原则是：每个产品动作都要对应到真实可执行的工作流能力，无法确认效果的部分进入实测矩阵。

## 总判断

本地工作流可以支撑 `techart-agent` 的产品形态，但第一版效果取决于两类模型能力：

- 图像编辑或参考图生成模型能否根据少量设定图生成新场景图。
- 传统扩散工作流能否通过参考图适配、ControlNet、img2img、inpaint 和 upscale 完成候选、精修、修复和导出。

普通文生图加文字 prompt 无法满足产品目标。它可以做补充，但不能成为主通路。

## 角色设计期资产生成

角色设计期最难，因为此时角色还没有稳定资产。产品希望从文字设定、草图和少量参考，生成面部标准图、全身图、服装图和细节图。

可行通路：

```text
图像编辑 / 参考图生成模型
  -> 根据草图、风格参考和文字设定生成候选资产

ControlNet lineart / sketch
  -> 根据草图或线稿约束轮廓

img2img
  -> 从用户草图或初稿继续变体
```

风险：

- 全身正面、侧面、背面的一致性难度高。
- 仅靠少量参考生成可靠三视图，成功率需要现场测试。
- 服装设定图的细节保留会波动。

产品策略：

- 第一版把设计期资产视为候选生成和筛选工具。
- 用户负责挑选可用资产。
- 通过资产进入角色库后，再用于后续延展和插画。

## 依赖资产延展

依赖资产延展比设计期更可控，因为已有面部标准、服装图、通过样例和角色比例。

可行通路：

```text
参考图适配
  -> 保持角色脸、画风和整体形象

ControlNet pose / lineart
  -> 控制姿态、动作和轮廓

img2img
  -> 基于已有资产做变体

inpaint
  -> 生成或修复表情、配饰、衣领、袖口、纹样
```

适合优先做：

- 表情差分
- 姿态参考
- 服装局部细节
- 已有图的轻微变体

风险：

- 大幅度视角变化会导致角色漂移。
- 复杂服装的背面和侧面推断成功率不稳定。
- 表情差分需要保持同一脸型，模型和参数要实测。

产品策略：

- 延展任务默认从已通过资产出发。
- 大改动进入候选筛选，小改动进入 img2img 或 inpaint。
- 通过图回写资产库，失败图写入质量反馈。

## 成品插画生成

成品插画生成是最适合第一版落地的模块。它不要求一次性生成完美图，产品可以通过候选、精修、局部修复和导出逐步推进。

可行通路：

```text
candidate_controlled
  -> 生成多张事件 CG 候选

ControlNet pose / lineart / depth
  -> 控制姿态、构图和空间关系

参考图适配
  -> 维持角色身份和服装倾向

img2img refine
  -> 提升选中候选图完成度

inpaint_region
  -> 修脸、手、配饰、衣服边缘和局部细节

export_upscale
  -> 输出项目规格图片
```

风险：

- 候选图中的服装细节可能丢失。
- 角色脸在大幅动作和复杂镜头中会漂。
- 手、眼睛、配饰和服装纹样需要局部修复。

产品策略：

- 一次生成多张候选。
- 让用户选中方向正确的图。
- 通过精修和局部修复提升到可用。
- 不把单次生成成败作为产品成败。

## 工作流优先级

第一阶段优先验证：

1. 图像编辑模型生成角色场景候选。
2. 参考图适配生成角色候选。
3. ControlNet 姿态控制。
4. img2img 精修。
5. inpaint 局部修复。
6. upscale 导出。

第二阶段再验证：

1. 多视图一致性生成。
2. 服装正侧背推断。
3. 表情表批量生成。
4. 角色 LoRA 训练数据整理。
5. 风格 LoRA 或项目级画风模型。

## 第一版不承诺的效果

第一版不承诺从一段文字直接稳定生成完整角色设定。

第一版不承诺少量参考图直接得到严格一致的三视图。

第一版不承诺复杂服装在所有角度都自动保持细节。

第一版不承诺一次生成就得到最终可交付图。

第一版承诺的产品能力是：组织资产、生成候选、人工筛选、精修、局部修复、导出和反馈沉淀。

## 验收方式

每个工作流 preset 都要有一组固定测试任务。测试任务使用同一角色、同一服装和同一事件，分别运行不同工作流组合。

验收结果进入表格：

```text
workflow preset
输入资产
输出数量
成功数量
角色一致性
服装保留
姿态遵循
局部错误
耗时
可继续加工比例
```

可继续加工比例比单张最好看更重要。产品面向生产效率，关键是让用户稳定得到可以继续加工的候选图。
