# 总仓库文档入口

这层文档负责总仓库入口和子项目分派。

当前总仓库只有两个子项目：

- [../domain-manage/README.md](../domain-manage/README.md)
  域名、Cloudflare、根域名静态页、子域名入口组织。
- [../openwaifu-agent/README.md](../openwaifu-agent/README.md)
  当前产品模块。

继续阅读时按这个规则走：

- 要看域名和公网入口组织，进 `domain-manage/`
- 要看产品代码、运行方式、环境配置、工作台和运维面板，进 `openwaifu-agent/`

总仓库根目录保留总入口说明和跨子项目约束。具体功能说明分别写在各子项目内部。
